# The test frame work

the test is a web framework for create sample crud for [Node.Js]("http://nodejs.org")

## instalation
create `package.json` and install with **npm**:
```console
npm i the-test
```

with **yarn**: 
```console
yarn add the-test
```

with **pnpm**:

```console
pnpm i the-test
```

## Features

* minimalist framework with (198KB) size
* big commiunity
